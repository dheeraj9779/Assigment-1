## Tasks Overview

### Task 1: Image Carousel with Theme Toggle
**Path:** `/` (Home page)

Interactive image carousel with dynamic flex-based expansion on hover. Features:
- 6-image carousel with smooth animations
- Light/Dark theme toggle with animated slider
- Dynamic flex sizing based on hover proximity
- Responsive design


### Task 2: Architecture Diagram
**Path:** `/task2`

Complex SVG-based system architecture visualization with animated connection paths. Features:
- SVG diagrams
- Tailwind CSS gradients
- Dashed line animations
- Dark theme with dot grid background

**Key Technologies:**
- SVG with stroke-dasharray
- Complex grid layouts
- CSS animations
